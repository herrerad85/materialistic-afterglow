# Assets

## Current

- `screenshot-1.png` to `screenshot-4.png`: app screenshots captured from a current debug build (Material 3 / Material You). Used by the project README and mirrored to the store listing under `fastlane/metadata/android/en-US/images/phoneScreenshots/`.
- `materialistic-hi-res-icon.png`: the 512x512 store icon, rendered from the app's current launcher artwork (the adaptive icon in `app/src/main/res`). The same image is used at `fastlane/metadata/android/en-US/images/icon.png`. It is a generated render of the live launcher icon, not a hand-authored brand master, and can be replaced in a future branding pass.

## Historical / pre-revival

These files predate the current design and are kept for reference only. They are not the source of the current screenshots or icon and should not be treated as up to date:

- `materialistic-drawable.sketch`
- `playstore-screenshots.sketch`
- `shortcuts.sketch`
- `materialistic-feature-graphic.png`

To refresh screenshots or the icon, capture from a current build or render from the launcher artwork in `app/src/main/res`, rather than editing these legacy files.
